    Section 1.1 Counting

    In these notes, the set of natural numbers \(\N\) is \(\{1,2,\ldots \}\text{.}\) For \(n\in \N\text{,}\) denote
    \begin{equation*} [n]=\{1,\ldots,n\}. \end{equation*}

    Definition 1.1.1.

    Let \(n\in \N\text{.}\) A permutation of \([n]\) is a bijective function \(\sigma:[n]\rightarrow [n]\text{.}\)

    Theorem 1.1.2.

    Let \(n\in \N\text{.}\) The number of permutations of \([n]\) is \(n!\text{.}\)

    Definition 1.1.3.

    The binomial coefficient \(\binom{n}{k}\) ("n choose k") is defined as the number of \(k\)-subsets of a set with \(n\) elements.

    Theorem 1.1.4 (Binomial Theorem).

    For \(n,k\in \N\cup \{0\}\text{,}\)
    \begin{equation} \binom{n}{k}=\frac{n!}{k!(n-k)!}=\frac{n(n-1)\ldots (n-k+1)}{k!}.\tag{1.1.1} \end{equation}
    When \(a\in \R, k\in \N\cup\{0\}\text{,}\) we extend the previous notation and define
    \begin{equation} \binom{a}{k}=\frac{a(a-1)\ldots (a-k+1)}{k!}.\tag{1.1.2} \end{equation}

    Theorem 1.1.5.

    For any \(x,y\in \R\) and \(n\in \N\text{,}\)
    \begin{equation} (x+y)^n=\sum_{k=0}^{n}\binom{n}{k}x^ky^{n-k}.\tag{1.1.3} \end{equation}

    Definition 1.1.6.

    A weak \(k\)-composition of \(n\in \N\) is a collection of integers \(a_1,\ldots,a_k\) such that
    \begin{equation} a_1,\ldots,a_k\geq 0 \text{ and } a_1+\ldots +a_k=n.\tag{1.1.4} \end{equation}

    Definition 1.1.7.

    A \(k\)-composition of \(n\in \N\) is a collection of integers \(a_1,\ldots,a_k\) such that
    \begin{equation} a_1,\ldots,a_k\geq 1 \text{ and } a_1+\ldots +a_k=n.\tag{1.1.5} \end{equation}

    Theorem 1.1.8.

    1. The number of weak \(k\)-compositions of \(n\) is \(\binom{n+k-1}{k-1}\text{.}\)
    2. The number of \(k\)-compositions of \(n\) is \(\binom{n-1}{k-1}\text{.}\)

    Definition 1.1.9.

    Let \(n,k\in \N\cup \{0\}\text{.}\)
    1. Denote by \(c(n,k)\) the number of permutations in \(S_n\) with exactly \(k\) cycles.
    2. The Stirling number \(s(n,k)\) of the first kind is defined as
      \begin{equation} s(n,k)=(-1)^{n-k}c(n,k).\tag{1.1.6} \end{equation}

    Theorem 1.1.10.

    1. \begin{equation} c(n,k)=(n-1)c(n-1,k)+c(n-1,k-1).\tag{1.1.7} \end{equation}
    2. For any \(x\in \R\)
      \begin{equation} \sum_{k=0}^{n}c(n,k)x^k=x(x+1)\ldots (x+n-1).\tag{1.1.8} \end{equation}
    3. For any \(x\in \R\)
      \begin{equation} \sum_{k=0}^{n}s(n,k)x^k=x(x-1)\ldots (x-n+1).\tag{1.1.9} \end{equation}

    Definition 1.1.11.

    Let \(n,k\in \N\cup \{0\}\text{.}\)
    1. Denote by \(P(n,k)\) the set of all the partitions of \([n]\) into exactly \(k\) parts.
    2. Define the Stirling number of second kind \(S(n,k)\) as the number of all partitions of \([n]\) into exactly \(k\) parts.

    Theorem 1.1.12.

    1. \begin{equation} S(n,k)=kS(n-1,k)+S(n-1,k-1).\tag{1.1.10} \end{equation}
    2. \begin{equation} \sum_{k=0}^{n}S(n,k)x(x-1)\ldots (x-k+1)=x^n.\tag{1.1.11} \end{equation}
    3. For fixed \(k\)
      \begin{equation} \sum_{n\ge k}S(n,k)\frac{X^n}{n!}=\frac{(e^X-1)^k}{k!}.\tag{1.1.12} \end{equation}

    Theorem 1.1.13.

    Let \(A\) and \(A_1,\ldots, A_n\) be finite sets. For \(\emptyset \ne J\subseteq [n]\text{,}\) denote
    \begin{equation} A_J=\cap_{j\in J}A_J,\tag{1.1.13} \end{equation}
    and \(A_{\emptyset}=A\text{.}\) Then
    \begin{equation} |A\setminus (A_1\cup \ldots \cup A_n)|=\sum_{J:J\subseteq [n]}(-1)^{|J|}|A_J|.\tag{1.1.14} \end{equation}

    Definition 1.1.14.

    Let \(n\in \N\text{.}\) A derangement of \([n]\) is a permutation \(\sigma:[n]\rightarrow [n]\) such that \(\sigma(a)\neq a\text{,}\) for any \(a\in [n]\text{.}\)

    Proposition 1.1.15.

    If \(n\in \N, n\geq 3\text{,}\) then
    \begin{equation} d_n=n(d_{n-1}+d_{n-2}).\tag{1.1.15} \end{equation}

    Theorem 1.1.16.

    Let \(n\in \N\text{.}\) If \(d_n\) is the number of derangements of \([n]\text{,}\) then
    \begin{equation} d_n=n!\sum_{j=0}^{n}\frac{(-1)^j}{j!}\approx \frac{n!}{e}.\tag{1.1.16} \end{equation}

    Theorem 1.1.17.

    If \(n,k\in \N\cup \{0\}\text{,}\) then
    \begin{equation} k!S(n,k)=\sum_{j=0}^{k}(-1)^{j}\binom{k}{j}(k-j)^n.\tag{1.1.17} \end{equation}

    Definition 1.1.18.

    The Euler totient function\(\varphi\) is defined as \(\varphi:\N\rightarrow \N\text{,}\) where \(\varphi(n)\) equals the number of natural numbers \(k\in [n]\) that are coprime with \(n\text{.}\)

    Theorem 1.1.19.

    If \(n\in \N, n\geq 2 \) has the prime decomposition \(n=p_1^{e_1}\ldots p_k^{e_k}\text{,}\) where \(k\ge 1, e_1,\ldots, e_k\ge 1\) and \(p_1,\ldots,p_k\) are distinct primes, then
    \begin{equation} \varphi(n)=n\left(1-\frac{1}{p_1}\right)\ldots \left(1-\frac{1}{p_k}\right).\tag{1.1.18} \end{equation}

    Definition 1.1.20.

    The classical Mobius function \(\mu\) is defined as \(\mu:\N\rightarrow \{-1,0,1\}\text{,}\) where \(\mu(1)=1\text{,}\) \(\mu(n)=0\) if \(n\geq 2\) is divisible by the square of a prime, and \(\mu(n)=(-1)^k\text{,}\) if \(n\) is the product of \(k\) distinct primes.

    Theorem 1.1.21.

    If \(n\in \N, n\geq 2\text{,}\) then
    \begin{equation} \sum_{d:d\mid n}\mu(d)=0.\tag{1.1.19} \end{equation}

    Theorem 1.1.22.

    Let \(f,g:\N\rightarrow \R\) be two functions. The following two statements are equivalent
    \begin{equation} (\forall n\in \N)(f(n)=\sum_{d:d\mid n}g(d)),\tag{1.1.20} \end{equation}
    and
    \begin{equation} (\forall n\in \N)(g(n)=\sum_{d:d\mid n}\mu(d)f(n/d)),\tag{1.1.21} \end{equation}

    Theorem 1.1.23.

    For a natural number \(n\) and a prime power \(q\text{,}\) let \(N(n,q)\) denote the number of monic irreducible polynomials of degree \(n\) in \(\mathbb{F}_q[X]\text{.}\) Then
    \begin{equation} N(n,q)=\frac{1}{2}\sum_{d:d\mid n}\mu\left(n/d\right)q^d.\tag{1.1.22} \end{equation}

    Definition 1.1.24.

    Let \(n\in \N\text{.}\) The Catalan number \(C_n\) is defined as the number of non-crossing triangulations of a convex \((n+2)\)-gon using \(n-1\) diagonals.

    Theorem 1.1.25.

    For any \(n\in \N\text{,}\)
    \begin{equation} C_{n+1}=\sum_{k=0}^{n}C_kC_{n-k}.\tag{1.1.23} \end{equation}

    Theorem 1.1.26.

    For \(n\in \N\text{,}\)
    \begin{equation} C_n=\frac{\binom{2n-2}{n-1}}{n}.\tag{1.1.24} \end{equation}

    Definition 1.1.27.

    For \(n\in \N\text{,}\) let \(p(n)\) denote the number of ways of writing \(n\) as the sum of natural numbers (order not important),
    \begin{equation} p(n)=|\{(i_1,i_2,i_3,\ldots): n=i_1+2i_2+3i_3+\ldots \}|.\tag{1.1.25} \end{equation}
    For convenience, define \(p(0)=1\text{.}\)

    Theorem 1.1.28.

    The generating function of \((p(n))_{n\geq 0}\) is
    \begin{equation} \sum_{n=0}^{\infty}p(n)X^n=\prod_{j=1}^{\infty}(1+X^j+X^{2j}+X^{3j}+\ldots)=\prod_{j=1}^{\infty}(1-X^j)^{-1}.\tag{1.1.26} \end{equation}

    Definition 1.1.29.

    For \(n\in \N\cup \{0\}\text{,}\) denote by \(u(n)\) the number of partitions of \(n\) into unequal parts.

    Definition 1.1.30.

    For \(n\in \N\cup \{0\}\text{,}\) denote by \(o(n)\) the number of partitions of \(n\) into odd parts.

    Theorem 1.1.31.

    1. The generating function of \((u(n))_{n\ge 0}\) is
      \begin{equation} \sum_{n=0}^{\infty}u(n)X^n=\prod_{j=1}^{\infty}(1+X^j).\tag{1.1.27} \end{equation}
    2. The generating function of \((o(n))_{n\ge 0}\) is
      \begin{equation} \sum_{n=0}^{\infty}o(n)X^{n}=\prod_{j=1}^{\infty}(1-X^{2j-1})^{-1}.\tag{1.1.28} \end{equation}
    3. For any \(n\in \N\text{,}\)
      \begin{equation} u(n)=o(n).\tag{1.1.29} \end{equation}
